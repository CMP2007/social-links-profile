## 📋 Technical Milestones of this Stage
In this specific phase of my training, I successfully achieved:

* **Flexbox Stack Architecture:** Implementation of a centralized layout using `display: flex` and `flex-direction: column` to manage the vertical alignment of profile elements and action buttons.
* **Component Styling & Overflows:** Use of `border-radius: 50%` combined with `overflow: hidden` to create a perfect circular avatar, a fundamental skill for profile UI components.
* **Dynamic Hover States:** Development of interactive feedback where the link buttons change background color, text color, and `font-weight` simultaneously, improving the user's affordance.
* **Typography & Contrast:** Integration of the 'Inter' font family with varying weights (`300`, `500`, `600`) to create a clear visual hierarchy between the name, location, and bio.



## 🛠️ Technologies (at the time)

* **CSS3:** Flexbox layout, media queries, and interactive states.
* **Normalize.css:** Used to ensure cross-browser visual consistency.
